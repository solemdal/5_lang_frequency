# Frequency Analysis of Words

# Start
$ python pprint_json.py '<path to file>'

# Example

IN: 
aa aa aa a a a a 
lol-kek

OUT:
Слово  a встречается в тексте  4
Слово  aa встречается в тексте  3
Слово  lol-kek встречается в тексте  1
