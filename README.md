# Frequency Analysis of Words

# Start
$ python lang_frequency.py 'path to file'

# Example
<br>
IN: <br>
aa aa aa a a a a <br>
lol-kek <br>
<br>
OUT: <br>
Слово  a встречается в тексте  4 <br>
Слово  aa встречается в тексте  3 <br>
Слово  lol-kek встречается в тексте  1 <br>
