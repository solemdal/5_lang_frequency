# Frequency Analysis of Words

# Start
$ python lang_frequency.py 'path to file'

# Example
<br>
IN: <br>
aa aa aa a a a a <br>
lol-kek <br>
<br>
OUT: <br>
Word a repeated in text 4 count <br>
Word aa repeated in text 3 count <br>
Word lol-kek repeated in text 1 count <br>
